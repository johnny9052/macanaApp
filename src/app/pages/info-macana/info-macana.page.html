¿Qué ofrece?
Macana es una aplicación móvil destinada al control y gestión de potreros en fincas, controlando el ganado, rotación de este y control de usuarios de dicha finca. Permite la asignación de permisos y gestión de usuarios para los accesos con privilegios.
¿Cómo accedo a la aplicación?
Si deseas tener una cuenta de prueba y una cotización escribenos al correo macanapotrerosapp@gmail.com o tambien para dudas y soporte técnico.
